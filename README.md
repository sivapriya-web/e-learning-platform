# 🎓 EduVerse - E-Learning Platform

EduVerse is a modern, responsive, and interactive front-end E-Learning Platform built using HTML, CSS, and JavaScript. The platform allows users to browse courses, watch learning videos, track their learning progress, manage their profile, and access a personalized dashboard.

## 🚀 Features

### 📚 Course Listing

* Browse available courses
* Course cards with descriptions and details
* Search courses by keyword
* Filter courses by category

### 🎥 Video Learning

* Embedded video lessons
* Dynamic course content
* Course-specific learning resources

### 📈 Progress Tracking

* Track lesson completion
* Dynamic progress bars
* Progress stored using LocalStorage
* Dashboard overview of learning progress

### 📊 Dashboard

* Overall learning statistics
* Course progress visualization
* Learning goals tracking
* Certificates section

### 👤 User Profile

* User information management
* Achievements section
* Settings panel
* Social media links

### 🌙 Dark Mode

* Light/Dark theme toggle
* Persistent theme using LocalStorage

### 📱 Responsive Design

* Desktop friendly
* Tablet responsive
* Mobile responsive
